IRIS v3.2.3 주요 배포내용 상세
IRIS 배포 안내 상세
[IRIS Studio]
개선
  • 기간 설정 / 시각화 옵션 > 시간 목록(프리셋)에 이전주, 이전분기 항목 추가
버그
  • 목록 화면 - 보고서를 다중 선택하고 권한 변경 팝업창을 띄우면 에러 발생
  • 목록 화면 - 필터 입력란에 일부 특수문자 입력시 에러 발생
  • (IE11) 편집 화면 진입시 웹 페이지 초기화가 대단히 느리거나 응답없음으로 처리되는 문제
  • 콤보박스 > 디폴트 값으로 초기화 되지 않는 문제
  • 콤보박스(Select-box) virtual scroll 오동작 문제
  • Chart / (IE11) Progress Bar 가 표시되지 않는 문제, 시각화 옵션 위치 버그
  • Chart / 테이블 - 행 선택 설정이 다중 선택일 때, 모든 항목을 해제하면 변수값이 갱신되지 않는 문제
  • Chart / 테이블 유형 - 객체 선택시 헤더 순서 설정이 초기화되는 문제
  • Chart / Windows Chrome 환경에서 차트에 글꼴이 적용되지 않는 문제
  • 날짜&시간, 기간 설정 / 트리거 설정시 코드 에러 발생
  • 데이터, 지도 객체 / 데이터 개수 제한 기능 버그
  • Popup, Portlet / 연결한 보고서 내에서 Nunjucks 문법이 정상 처리되지 않는 문제
  • 반응형 모드 height 관련 오동작 문제
[IRIS Map Analyzer]
버그
  • 주소 검색시 버튼 클릭이 되지 않는 문제
[IRIS Web Platform]
버그
  • 회원가입시 패스워드 처리 오류 문제

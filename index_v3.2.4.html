IRIS v3.2.4 주요 배포내용 상세
IRIS 배포 안내 상세
[IRIS Studio]
신규
  • 목록 화면 / 목록 보기 타입의 기본값을 설정하는 기능 추가
  • 데이터 / DB - Advanced Query 입력 기능 추가
개선
  • 목록 / 초기 출력 컨텐츠 개수를 30개 → 60개 로 변경
  • 보고서 등의 리스트뷰에서 컨텐츠 실행을 더블 클릭 -> 원 클릭으로 변경 제안
  • 객체 입력모드 전환 버튼이 다른 객체에 가려지는 문제
  • Chart / 테이블 유형 - 조건부 서식에서 다양한 조건 수행
  • Chart / 테이블 유형 - 조건부 서식을 '행'에 적용시 border 가 사라지는 문제
  • Chart / 테이블 유형 - 헤더 설정의 순서 기능을 OFF 하는 옵션 추가
  • Chart / 막대형 - 시각화 옵션 > 스택 반전 옵션 추가
  • IMA 뷰어 / 편집시 저장된 지도 중심 위치/줌레벨이 유지되도록 개선
  • 날짜&시간, 기간 설정 / 설정된 변수값이 포맷과 맞지 않을 때 입력란을 비우도록 동작 변경
  • Popup, Portlet / 보고서 불러오기 실패시 에러 표현 or 메시지 개선
  • 이미지 객체 / 성능 개선
버그
  • 보고서 가져오기 / 내보내기URL hashcode 는 덮어쓰기 하지 않는 문제
  • 데이터 객체 / DB 유형 - 내보내기URL 로 보고서 실행시 요청 URL 오류
  • 데이터, 지도 객체 / 데이터 개수 제한 기능 버그
  • Chart / 테이블 유형 - 시각화 옵션 > 헤더 > 일괄 설정의 콤보박스가 닫히지 않는 문제
  • Chart / 양방향 막대형 - 시각화 옵션 > 탭 헤더 텍스트 오류 (X축, Y축이 반대로)
  • 기간 설정 / (편집 화면) 프리셋을 선택해도 입력/선택 UI에 반영되지 않는 문제
  • 조건 검사 / 변수명이 변경되지 않는 문제
  • Check&Radio / 항목 배열 방향 - 세로 설정시 가로 스크롤 발생
[IRIS Map Analyzer]
신규
  • 목록 화면 - 목록 보기 타입의 기본값을 설정하는 기능 추가
  • LINESTRING 신규 시각화 타입 개발
버그
  • 데이터가 지도에 표시되고 있을 경우 basemap 변경 불가
  • Leaflet 에서 basemap 변경 시 반응이 느림
  • 주소검색시 특수문자 넣으면 에러출력됨
  • DSL 실행 오류가 발생 했지만 데이터 조회 화면이 멈추지 않습니다.
[IRIS Web Platform]
신규
  • 환경설정 / Studio, IMA 목록 보기 타입 설정 기능
  • 상단 팝업 dashboard 바로가기 버튼 표시 설정
  • 환경설정 / 도움말 버튼 사용 여부 설정 기
  • 환경설정에 Sentry 설정 추가
  • IRIS Support 표시여부 설정 추가

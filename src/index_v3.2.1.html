IRIS v3.2.1 주요 배포내용 상세
IRIS 배포 안내 상세
[IRIS Studio]
개선
  • Chart / 막대형 차트의 막대(bar)의 색을 개별 지정할 수 있는 기능 개선
  • Map / 데이터 설정에 "캐시 설정", "데이터 개수 제한" 옵션이 없는 문제 개선
  • 편집 화면에서 컴포넌트 선택시, 컴포넌트 유형에 대해 알 수 있도록 UI 개선
  • 날짜&시간 / 초기 날짜 설정 - 상대 시간으로 설정시 처리되는 값 오류 개선
버그
  • 보고서가 썸네일 뷰 일 경우 추가 보고서가 로딩 되지 않는 문제
  • 보고서 편집 > 다른이름으로 저장 시 이름 중복으로 저장되지 않는 문제
  • 데이터 > 데이터 모델 - file 유형의 상세보기 버튼 클릭 시 검색어가 null 으로 표시되는 문제
  • Map의 데이터 모델 정렬이 바로 적용이 되지 않는 문제
  • Label > 조건부 서식이 적용된 뒤, 원래대로 돌아오지 않는 문제
  • 편집 화면 - 우측 패널 미리보기 창 스타일 깨짐
  • 객체 네비게이션 기능이 반대로 작동하는 것 같습니다.
[IRIS Map Analyzer]
개선
  • 데이터 추가 > DB > DSL 명령어 기능 제한 안내 툴 팁 적용
  • 데이터 추가 > DB > DSL 안내 툴팁 도움말 링크가 마우스 오버시 깜빡이는 이슈
  • button 엘리먼트를 ui-button 컴포넌트로 개선
[IRIS Web Platform]
개선
  • 메뉴 개수에 따른 UI 개선
버그
  • 메뉴 개수에 따른 UI 개선 - IE/Edge 이슈 처리
  • 자동 로그아웃 후에 간헐적으로 메뉴 관련 오류 메세지 발생 하는 문제
  • 환경설정 > 기본설정 > 일반설정 > 로그인 사용자 정보 > 대표정보 변경 예외 처리 추가
  • 환경설정 > 기본설정 > 일반설정 > 로그인 사용자 정보 > 대표정보 사용자 입력 필드 "|" 문자 입력 시 값이 이상하게 표시됨

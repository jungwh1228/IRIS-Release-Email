IRIS v3.1.1 주요 배포내용 상세
IRIS 배포 안내 상세
[IRIS Studio]
신규
  • 슬라이더 - 동적 구성, 눈금 관련 기획 변경 사항 재개발
  • 보고서 소유자 변경 기능
  • Tab / 텍스트 스타일 설정 기능 추가
  • 인쇄용 캔버스 크기, 인쇄 버튼 추가
  • 신규 컴포넌트(객체) 추가 - 구분선개발
개선
  • Tab / 값 대체 설정 안내 툴팁의 설명 오류
  • 다운로드 기능을 숨기는 서버 설정 추가
  • 슬라이더 / 컴포넌트 동작 개선
  • 트리거 설정 목록의 "선택 이동" 버튼 아이콘 수정 요청
  • Chart / 히트맵, 트리맵 - 시각화 옵션에 범례 위치 옵션 추가
  • Chart / 선택한 시각화 유형 아이콘을 객체 배경에도 적용
  • 일부 객체에 꾸미기 옵션을 추가
  • Chart / 꺾은선형 - 대시 스타일 설정 기능 추가
  • Chart / 신규 차트 유형에 '축 반전' 옵션 추가
  • Chart / Gauge 유형 - Pene 크기 설정 추가
  • Chart / 차트 크기 (객체 맞춤/사용자 설정) 설정 추가
  • 데이터 객체 / 선택한 데이터 유형의 아이콘을 객체에도 적용
  • 전체 변수명 보기 창에 정렬, 필터 기능 추가
버그
  • Chart / 배경을 투명하게 하였을 때, 투명이 아닌 배경 이미지가 표시되는 문제
  • 텍스트변환 객체의 명령어 입력창의 크기 조정이 안됩니다.
  • 신규 보고서 생성시 Label 텍스트 클립보드 복사 안됨
  • IE 에서 일부 UI 깨짐
  • v3.1.0-RELEASE 배포 UI 이슈
  • 시각화 옵션 Dialog(팝업) 드래그 시 color-picker는 같이 이동하지 않는 문제
  • 데이터 객체 / DSMS 유형일 때, 다른 객체를 선택해도 연결정보 유형 선택이 유지되는 문제
  • Chart / 편집화면에서, 탭에 의해 숨겨진 상태일 때 데이터를 수신했어도 차트가 그려지지 않는 문제
  • 보고서 '화면 이미지 캡쳐' 기능 사용시, 보고서 하단이 잘려서 저장되는 문제
  • IMA 뷰어 / '지도 이동' 모드일 때, 우클릭 시 context menu 가 나타나지 않는 문제
  • Chart / 반응형일 때 캔버스 크기 변경 시 차트 Redraw 처리
  • 편집 화면에서, 데이터를 수신하는 중에는 데이터와 관련된 UI를 비활성화 처리
[IRIS Map Analyzer]
신규
  • Pandas 연동 및 데이터 조회 기능 강화
  • 공간데이터셋 타입(geometry) 추가
  • DSL 명령어 이력 표시
  • DSL 명령어 도움말
개선
  • 프로젝트 목록 썸네일 개선
  • https 관련 이슈 가
  • 편집창 레이어 리스트 개선
  • 프로젝트 보기모드 > 레이어 리스트가 비었을 때 출력되는 리스트가 오류처럼 보입니다.
  • 드롭다운 다중 선택 시 이전 드롭다운이 닫히지 않는 문제
  • 다운로드 기능 개선 (geojson)
  • 디자인 개편 적용
  • 데이터 모델 설정창 개선
  • 레이어 이름 수정 로직 개선
버그
  • basemap 지도 설정이 변경되었을 경우, 기존 지도에 변경된 값을 반영하지 못하는 문제
  • 3.1.X 버전에서 예전 생성한 IMA 프로젝트에서 일부 레이어가 보이지 않습니다.
  • 주제도 마커 편집 오류 (mapbox)
  • 공간데이터셋 geojson 데이터는 geometry 로 경계선은 그리지만, 라벨, 채우기 색상, 임계치 적용, 툴팁이 표시 적용되지 않습니다.
  • 섬네일뷰 > 컨텐츠 관리의 툴팁 불일치
  • 프로젝트 "내보내기 링크" 팝업창에서 링크 생성을 하면 링크 복사를 위한 클릭이 비 활성화된 채로 링크URL이 생성됩니다. 안내문구는 클릭하라고 나오지만, 저장버튼을 누른 후에야 링크복사가 가능해집니다.
  • 리스트뷰 > 카테고리명이 길 경우, 앞 뒤가 잘리는 문제
[IRIS Web Platform]
신규
  • 신규 스튜디오 연동 기능 추가
버그
  • 인증 후, 다른 사용자로 SSO 인증을 시도하면 이전 사용자로 세션이 살아 있는 문제
[IRIS Analyzer]
개선
  • 데이터 모델 / 새 모델 뷰 - 검색어 입력란 크기를 조절 가능하도록 수정
  • 컬럼 목록 - 마우스 우클릭 메뉴에 모든 컬럼명을 클립보드 복사하는 메뉴 추가
  • 검색어 추천 기능 - 도움말 내용을 API 연동으로 변경
  • 다운로드 기능을 사용할 수 없도록 하는 설정 추가
버그
  • 검색 / 데이터 모델 - 샘플링 설정에서 임의의 '샘플링 방식' 선택 후, 취소로 창을 닫을 경우 '샘플링 방식'이 이전에 선택한 것으로 남아있는 문제
[IRIS Bigdata Platform]
개선
  • [DB Browser] 데이터베이스 선택 콤보박스에 입력 필터(에디터) 기능 추가
  • [DB Browser] 다운로드 기능 비활성화 옵션 추가
  • [File Browser] 파일 이동 - 파일명에 % 기호가 포함된 경우 에러 발생
  • [IBP] 각 데이터 모델 목록 UI 마다 정렬 기준이 다른 문제
버그
  • [File Browser] HDFS super user 가 root 가 아닐때 문제
  • [File Browser] MinIO / 이미지 링크 복사 기능 오류
  • [IBP] 사용자 관리 > 그룹 / 목록 No. 필드 오류
  • [데이터 추가] 데이터 입력 설정하기 단계에서 컬럼 유형이 설정되지 않는 버그
